This is a simple Sudoku App. It has a backtracking solver, keyboard/mouse functionality, and dynamic resizing. All of the graphics are done through CMU graphics.

Start:
-To start the app just run the main.py file

Libraries:
-CMU Graphics (CMU graphics is needed for all of the display functionality)

Notes:
-*IMPORTANT* CMU graphics does not work in python 3.12, ensure you are using 3.11 or earlier
	-this may be updated in the future, check this link: https://academy.cs.cmu.edu/desktop
-ensure the board files is in the same directory as the .py files
-All keybinds can be found on the help screen
